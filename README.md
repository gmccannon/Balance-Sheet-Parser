# Balance-Sheet-Parser
This is a random attpemt at a stock price prediction model by analyzing various information found in a stock's financial documents. The output is a floating point number representing a multiple of the predicted price of the stock one year from now, versus the current price. (i.e. a value of 1.5 represents a prediction that the stock price will increase 50% in the following year, and a value of 0.4 is a prediction that the stock will decrease 60% in the following year). The process incudes grabbing financial documents and training a model, thus the repo also doubles as a parser for financial information.
